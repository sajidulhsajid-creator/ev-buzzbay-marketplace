# BuzzBay - EV Charging Marketplace

Find and book EV chargers in Australia!

## Live Demo
https://YOUR-USERNAME.github.io/ev-buzzbay-marketplace/

## Features
- Real-time charger availability
- Live LoRa mesh network data
- Instant booking
- Navigation with Google Maps
- Distance & driving time

## How to Use
1. Click "Use My Location" to enable geolocation
2. Browse available chargers on the map
3. Click a charger to see details
4. Book a charging session
5. Get driving directions

## Built With
- Leaflet (Maps)
- HTML5 + CSS3 + JavaScript
- Real-time MQTT data from LoRa mesh
